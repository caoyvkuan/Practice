# 句子的组成

+ 主语、谓语、宾语、表语、补语、定语、状语、同位语

---

## 主语

+ 主要叙述的 **对象** 或是 **内容** -> 一句话的主体
+ 可以做主语的词语
  + 名词、代词、数词、动词不定式、动名词、句子、其他
  + 名词 -> 某东西的名称
  + 代词 -> 在句子中代替名词,避免主语名词重复出现
  + 人称代词
    + 主 - 宾
    + I - me
    + you - you
    + she - her
    + he - him

## 谓语

+ 说明主语 **动作** 或 **状态** 的词语

+ 也分 动作型谓语 和 状态型谓语

+ 说明主语情况就是谓语

+ 动作型谓语
  
  + 表示动作(看得见的动作) -> eat(吃) drink(喝) play(玩) buy(买)
  + 心理行为(看不见的内心活动) -> think(思考) link(喜欢) know(知道) consider(考虑) hope(希望)
  
  ---
  
  + 简单谓语
    + 实义动词 - 有实际意义的动词 如: bathe(洗澡)
    + 动词短语 - 由一个单词或几个单词构成的词组 如: look after(照顾)
  + 复合谓语
    + 简单谓语在加上其他词语共同作谓语的情况
    1. 情态动词 + 实义动词 或 动词短语的原型
    + I speak Chinese. -> I can speak Chinese. -> **can speak** 就是复合谓语
    + I can look after her. -> **can look after**
    2. 助动词 + 实义动词 或 动词短语的原型
       + I __but__ a book every year
       + I __will buy__ a book next year
    3. 助动词 + 其他动词形式(时态 - 过去式)
       + I __am working__ now.

+ 状态型谓语
  
  + __系动词 + 表语__  -- __be + 表语__
  + be 有八种不同的形式，意思都相同
    + _be 、 am 、is 、are 、was 、were 、being 、been_
    + 常用的有 _is 、 am 、 are_
  + I __am happy__
  + He __is a doctor__
  + You __are cool__

## 表语(主语补语)

+ 表语属于 谓语的范畴, 因为 __系动词 + 表语__ 是谓语
+ _系动词和表语_ 是一起出现的
+ _系动词_ 后面的就是 _表语_
+ _名词、形容词、介词短语、名词所有格、动词不定式、动名词、代词、数词、分词、副词、句子_ 都可以做表语
+ 形容词 , 名词 , 介词短语 例子
  + I am __rich__
  + She is a __nurse__
  + I am __in GuanXi__ -> _in..._ 介词 在什么地方

## 宾语

+ 动作的 __对象__ 或 __承受者__
+ 实义动词分为 _及物动词_ 和 _不及物动词_
+ 及物动词 -> 涉及到其他事物的动词 如: 买 _东西_(这个买的东西就是 _宾语_)
  + buy(买) borrow(借) give(给)
+ 能做主语的词语就可以做宾语
+ 人称代词做宾语必须用宾格
+ 特殊的及物动词 - 双宾 -> 间接宾语(人) + 直接宾语(物)
  + I gave _her_ a _book_ (我给她一本书)

---

+ 宾语补足语 - 宾补
  
  + 可以做宾补的有 : 形容词、名词、数词、动词不定式、动名词、副词、分词、介词短语、句子
  + 加上一个词语补充宾语的意思,使句子完整
  + I make her (我让她,我使她) 句子不完整
  + I make her **happy**, 这样才是完整的 happy 就是宾补
  + He made _the English class_ **funny**

+ I gave _her_ **a book** -> 双宾 -> 间接宾语(人) + 直接宾语(物)

+ I call _her_ **Mary** -> 宾补 -> 两个指的是同一个人或物,所以是宾补

## 状语

+ 说明谓语的情况,只跟**谓语**有关系
+ 多数状语放句子后面
+ 一句话中状语可以出现多个
+ 分类 -> 时间、地点、方式、原因、目的
  + in Beijing -> 地点状语
  + I _teach_ English __in Beijing__
  + I _am happy_ __in Hubei__
  + slowly 方式状语
  + I _speak_ English __slowly__
  + I __usually__ _go_ home __after work__

## 定语

+ __修饰或限定__ __名词+代词__ 的词语叫定语
+ 限定词语的范围
+ 普通名词才能带定语,专有名词一般特殊情况才带定语
+ 分类 -> 前置定语 | 后置定语
+ 前置
  + 形容词性物主代词、形容词
  + 名词所有格、量词、指示代词、名词、数词
  + __English__ book -> 用英语修饰书
  + __Two__ __beautiful__ girls -> 多个定语
+ 后置
  + 介词短语、名词所有格(of + 名词)、动名词短语
  + 动词不定式短语、句子

## 同位语

+ 两个不同的词语都表示同一个事物
+ 同位语起到强调补充说明的作用
+ __my father__, __the fat man__, is drinking -> 这两个词都表示同一个人

# 句型

+ 状态型句子
  
  + 主语 + be(系) + 表语 + 状语

+ 动作型句子
  
  + 主语 + 谓语 + 宾语 + 状语

+ 造句首先确定 谓语和状语 , 然后去找主语和宾语 , 避免中国式英语

## 1. 主语 + 谓语 [+ 状语]

+ 主语 + 谓语(不及物动词) + 状语(表示动作)

+ I work every day. 我每天工作
  
  + __work__ 不及物动词做谓语
  + __every day__ 时间状语

+ I live in DaYe city. 我住在大冶市
  
  + __live__ 不及物动词
  + __in DaYe city__ 地点状语
  + 不能直接加地名,如 I live DaYe.
  + live 是不及物动词所以不能直接加宾语

## 2. 主语 + be + 表语 [+ 状语]

+ 主语 + be(系动词) + 表语 + 状语(表示状态)

+ 特殊的主系表结构
  
  + 主语 + 特殊的系动词 + 形容词(表语)
  + look
  + get
  + feel
  + You(主语非三单) look beautiful(形容词不能做宾语) 所以 look 变成特殊系动词
  + 主语三单格式,  系动词也变三单格式

+ She is rich now. 她现在很富有
  
  + __is rich__ 系表结构谓语 - 状态型谓语
    + rich 形容词做表语
  + __now__ 时间状语

+ I was a __worker__ __before__

## 3. 主语 + 谓语 + 宾语 [+ 状语]

+ 主语 + 谓语(及物动词) + 宾语 + 状语(表示动作)

+ I read English every morning. 我每天早上读英语
  
  + __read__ 及物动词(谓语)
  + __English__ 宾语
  + __every morning__ 时间状语

+ She learns English very hard. 她很努力学英语
  
  + __learn__ 及物动词
  + __hard__ 方式副词做状语

## 4. 主语 + 谓语 + 双宾语 [+ 状语]

+ 主语 + 谓语(及物动词里面的双宾动词) + 双宾语 + 状语(表示动作)

+ I gave her a book yesterday. 我昨天给了她一本书
  
  + __gave__ 谓语
  + __her__ 间接宾语
  + __book__ 直接宾语
  + __yesterday__ 时间状语

+ They sent me a computer last week. 上周,他们送给我一台电脑
  
  + __sent__ 谓语
  + __me__ 间接宾语
  + __computer__ 直接宾语
  + __last week__ 时间状语

## 5. 主语 + 谓语 + 宾语 + 宾补 [+ 状语]

+ 主语 + 谓语(少数特定的及物动词) + 宾语 + 宾补 + 状语(表示动作)

+ I made her happy yesterday. 昨天,我让她很开心
  
  + __made__ 及物动词作谓语
  + __her__ 宾语
  + __happy__ 形容词做宾补
  + __yesterday__ 时间状语

# 名词 n = nouns

+ 名词的定义,各种东西的名称都叫**名词**

+ 名词分类
  + 普通
  + 专有

+ 不可数名词, 在特指的某些情况下也是可数的
  + 专有名词同样如此

+ 复合名词 -> 名词(不能变化) + 名词(决定 可数 或 不可数)
  
  + English book(英语书)
  + school gate(学校大门)
  + Room number(房间号码)

+ **可数名词单数不能单独用**

+ 复数虽然可以直接用, 但在某些不想泛指的情况同样需要加限定词

+ 不可数名词也可以直接用, 但同样表示泛指,可能需要限定词

## 普通名词

+ 可数 -> __c = countable noun__
  
  + 一般为能够分清数量的，如: 书、手机
  + 一个可数名词的单数**不能单独在句子或短语中**出现,需要加**限定词**
  + 限定词一般放在修饰名词的词语前面

+ 不可数 -> __u = uncountable noun__
  
  + 一般无法分清数量的,如: 水、牛奶、米、头发
  + 不可数名词没有复数形式
  + **量词 + 不可数名词** -> two cups of water

+ 量词构成 - 量词中的 of 没有意思
  
  + 不是所有名词都可以构成量词
  + 一般为容器或有内部空间的词
  + 符合逻辑的

+ A/an/one + 可数名词单数 + of
  
  + a cup of
  + a bucket of

+ 数词 + 可数名词的复数 + of
  
  + two bottles of (两瓶)
  + three tins of (三罐)

+ 量词也可以修饰**可数名词的复数**

## 专有名词

+ 人名、地名、各种机构、组织、术语
  + 专有名词可以单独使用
  + 专有名词 __首字母大写__

## 复数形式

+ 26个字母中除了5个元音字母 __A、E、I、O、U__ 其他都是辅音字母

+ 大部分名词变复数 -> __+ s__
  
  + pen(钢笔) -> pens
  + cup(杯子) -> cups

---

+ 当可数名词以字母 __s 、sh 、x 、ch__ 结尾 -> __+ es__
  
  + box(箱子) -> boxes
  
  + kiss(吻) -> kisses
  
  + watch(手表) -> watches
  
  + brush(刷子) -> brushes
    
    > 例外 : stomach(胃) -> stomachs

---

+ 以辅音字母 + y 结尾 -> 改 __y 为 i__ __+ es__
  + city(城市) -> cities
  + baby(婴儿) -> babies
  + country(国家) -> countries
  + dictionary(字典) -> dictionaries
+ 元音字母 + y 结尾 -> 直接 + s
  + toy(玩具) -> toys
  + boy(男孩) -> boys
  + day(天) -> days

---

+ 以字母 f 或 fe 结尾 -> 改 __f 或 fe 为 v__ __+ es__
  
  + wife(妻子) -> wives
  
  + knife(小刀) -> knives
  
  + thief(小偷) -> thieves
    
    > roof(房顶) -> roofs
    > gulf(海湾) -> gulfs
    > cliff(悬崖) -> cliffs

---

+ 辅音字母 + o 结尾 -> __+ es__
  + tomato(西红柿) -> tomatoes
  + potato(土豆) -> potatoes
+ 元音字母 + o 结尾 -> __+ s__
  + video(视频) -> videos
  + studio(工作室) -> studios

---

+ 不规则变化
  + man(男人) -> men
  + mouse(老是) -> mice
  + goose(鹅) -> geese
  + ox(牛) -> oxen
  + child(小孩) -> children
+ 还有单复数相同的
  + Deer(鹿)
  + Chinese(中国人)
  + sheep(绵羊)
+ 复数名词 -> 没有单数形式
  + People(人们)
  + police(警察)
  + public(公众)

## 所有格

+ 表示所属关系的一种格式
  
  + 小明的书包 -> __小明的__ 就是所有格
  + 因为书包是属于小明的

+ 构成方式
  
  + 有生命的名词 + __'s__ , 一般指人或动物
    + 名词以 s 结尾直接 + __'__, 可以省略 s
    + 一般人名都可以直接 + 's
      + NY's
    + 普通名词单数不能单独使用
      + My friend's (需要加限定词 My 才可以构成所有格)
      + The Doctor's
    + and 表示共有, 只需后面加 s
      + N and Y's bed - N和Y 共有的床
    + and 表示各自, 都要加 s
      + N's and Y's beds - N和Y 各自的床
  + 没生命的名词用 __of + 无生命名词__, 表示名词的性质特征
    + 可数名词不可直接用, 需要加限定词
    + _the_ center __of the city__(城市的中心)
    + 所有格做后置定语-修饰的-名词-都需要加-定冠词(_the_ prices)
    + of the house(房子的) _the_ prices of the house is 500$

+ 名词所有格作前置定语可以修饰 可数名词和不可数名词
  
  + Lily's tea
  + My sister's friends
  + The dog's food

## 人称代词

+ 主格 - 宾格
  
  + 我 - I - me
  + 我们 - we - us
  + 你/你们 - You - you
  + 他 - he - him
  + 她 - she - her
  + 它 - it - it
  + they - them - 他们 - 她们 - 它们

+ 主格做主语,不能做宾语

+ 宾格做宾语,不能做主语

+ 都可以做表语

+ 两个以上人称代词做主格的顺序(先说别人在说自己)
  
  + you, she(he) and I
  + we, you and they
  + He and I are both form DaYe;
  + She and I like money.

## 物主代词

+ 表示所有关系的代词

+ 形容词性物主代词 - 名词性物主代词
  
  + my - 我的 - mine
  + your - 你的/你们的 - yours
  + her - 她的 - hers
  + his - 他的 - his
  + its - 它的 - its
  + our - 我们的 - ours
  + their - 她们的 - 他们的 - 它们的 - theirs

+ 名词性物主代词例子 - 主语 - 表语 - 宾语
  
  + 这是 *我的* 书 - This is my book.
    + 这里表示 *书* 是 *我的* 所以用 my
  + 你的书比 *我的* 贵
    + Your book is more expensive than mine.
    + 这里的 *我的* 是指的 *我的书*, 而不是自己 所以用 mine
  + 你的就是我的,我的还是我的.
    + Yours is mine, mine is still mine
  + 不能重复使用
    + your book is better than yours. 错的
    + yours 在英语中不能一个人比较
    + yours 改为 his 或其他才行

+ 形容词性物主代词
  
  + 只有一个作用,就是作定语,且是前置定语
  + 没有名词就不可能出现
  + 可数不可数名词都可以修饰,修饰名词不需要加冠语
  + my house
  + My houses

## 指示代词

+ 可以作 : 主语 , 宾语 , 表语 , 定语
+ 单数 - 复数
  + this(这个) - these(这些) - 指示较近的人或物
  + that(那个) - those(那些) - 指示较远的人或物

+ 单数形式可以修饰单数,也可以修饰不可数名词
  + This computer 这台电脑
  + This water 这水 / 这些水
+ 复数形式,只能修饰可数名词复数
  + These houses
  + Those boys

+ 相互,彼此

  + We should help __each other__.

## 不定

+ 用以描述, 不确定的人/物
+ 形容词修饰不定代词要后置

+ many - 许多, 修饰可数名词复数
+ much - 许多, 修饰不可数名词
+ a few - (肯定)一些, 可数
  + few - (否定)几乎没有
+ a little - (肯定)一些, 不可数
  + little - (否定)几乎没有
+ 一些
  + some - 用在肯定句, 想要正面肯定的回答时, 可以用在疑问句
    + Would you like some coffee?
  + any - 用在否定句/疑问句, 表示任何一个时, 可以用在肯定句

+ each / every - 表示每, 后接单数
  + each carrot / every carrot
  + each 可以做代词, 单独做主语, 两个或以上的每一个
  + every + 名词, 不可单独做主语, 三个或以上的每一个

+ either - 两者中的每一个, 谓语用单数
+ either - 两者中每一个都不
  + Either of the carrots is good.

+ 都, 全部
  + both - 两者 都
  + all - 三者及以上 都

+ other + 名词 - 另外的, 其他的 
+ the other - 特指两者中的另一个
+ another - 泛指三个及以上的另一个

+ one, it - 代指

+ 复合不定代词
  + everyone / everybody / everything
  + someone / somebody / something
  + anyone / anybody / anything
  + no one / nobody / nothing

## 疑问

+ 只带未知的人/物
+ 谁: Who - whom - 主格与宾格的区别
  + Who saw the rabbit? - 谁看见了兔子?
  + Whom did the rabbit see? - 兔子看见谁了?
  + 一般情况也可以用 who 取代 whom, 除了做介词宾语时

+ what - 什么
+ which - 哪一个
+ whose - 谁的

  + what - whatever

# 形容词 adj = adjectives

+ [形容词](https://www.bilibili.com/video/BV1ja4y1e7KV?p=13)

+ 主要用来修饰**名词或代词**

+ 可以做 : 前置定语 , 表语 , 宾补

+ 普通形容词
  
  + big(大的)
  + green(绿的)
  + happy(快乐的)

+ 复合形容词
  
  + sky-blue(天蓝色的)
  + good-looking(好看的)
  + man-made(人造的)

+ 多个形容词的摆放顺序,用 and 也可以不用顺序
  
  1. 冠词(a,an,the 物主代词 my, 数词 three,指示代词等)
  2. 描绘性的形容词 beautiful
  3. 表示形状(大小,长短,高矮)的形容词
  4. 表示年龄或者新旧的形容词
  5. 表示颜色
  6. 表示出处来源
  7. 表示材料,物质的形容词
  8. 表示用途

+ __a beautiful,tall,young__ girl. 一个年轻漂亮,身材高挑的女孩子

+ my __nice small brown leather__ bag. 我漂亮的棕色小皮包

## 作表语

+ 大部分形容词都可以做表语
+ 造句 : 主语 + be(系动词) + 形容词(表语)
  + you are __rich__
  + she is __busy__

+ 一些以 a 开头的形容词, 只做表语, 不能做前置定语
+ 但有时可以做后置定语
  + alive, asleep, alone
  + 做定语替换为: living, sleeping, lonely
+ 某些表示身体状况的同样如此
  + 表语: ill / well , 定语: sick / healthy

## 做前置定语

+ 大部分都可以做前置定语
+ 可以构成一个*短语*,不能直接造句
+ 多个形容词顺序
  + 限定词 特征 尺寸 形状 新旧 颜色 来源 材料 名词

+ The __big__ house

+ My __small__ house

+ __cheap__ computer

+ I like beautiful girl 错
  
  + 名词有形容词也需要加冠词
  + I like the __beautiful__ girl
  + I like the girl. 不需要形容词也是个完整的句子
  + 形容词只是附加成分, 哪里有名词哪里就有定语

  + the __rabbit (that is) busy eating carrots__

## 复合形容词

  + 白发的: a white-haired man = a man whose hair is white

## 比较级和最高级

+ 用法和普通形容词一样
  + 在比较的时候需要变比较级
  + 比如 : 什么比什么漂亮
+ 如 : 
  + 漂亮(原级)
  + 更漂亮(比较级) - more
  + 最漂亮(最高级) - most
+ 一般结尾直接 + __er/est__
  + tall(高的) - taller(更高) - tallest(最高)
  + gentle(温柔) - gentler(更温柔) - gentlest(最温柔)
+ 有些以 e 结尾的词 + __r/st__
+ 辅音字母 + y 结尾, + __y 变 i + er/est__
+ 重读闭音节, __+ 双写词尾辅音字母 + er/est__
+ 多音节/部分双音节词, __词前 + more/most__
+ 不规则
  + good - better - best
  + bad - worse - worst
  + many - more - most
  + little - less - least

+ 比较: bigger than ...
+ 倍数: three times bigger than ...
+ 一样: as...as... / three times as...as...
+ 越来越: prettier and prettier / more and more beautiful
+ 越..., 就越...
  + the + 比较级(原因) + 补充内容, the 比较级(结果) + 内容
  + The more carrots the rabbit eats, the smarter he will become
+ 省略: 依靠语境 - The more, the better - 越多越好

+ 最高级: 一般用 __in + 地点__ 或 __of + 所属范围__ 限定范围
  + 形容词前加序数词表示 第几最
  + 形容词最高级修饰名词, 前面一般都 + the , 表特指
  + I am most NB __in my home__
  + I am the most excellent __one of my classroom__

# 动词 v = verbs

+ 不及物动词 -> __vi = intransitive verb__

+ 及物动词 -> __vt = transitive verb__

+ 助动词 -> __aux.v = auxiliary verb__

+ 情态动词 -> __model verb__

+ 表示动作或心理行为的词语

+ 按照用法可以分为 谓语动词 和 非谓语动词

+ 谓语动词做谓语

+ 非谓语动词可以作: 主语,宾语,宾补,表语,定语,状语

## 祈使语气

+ 多用于 命令/请求
+ Rabbit, eat the carrot!
+ Please, give me thumbs-up!

+ 窍门
  + 用第二人 "你" 用一般现在时态
  + 去掉 "你", 并保证动词原形

+ You eat the carrot! - Eat the carrot!
+ You don't eat it! - Don't eat it!
+ You are quiet! - be quiet!
+ You never give up! - Never give up!

## 使役

+ 通常表达如 `使, 让, 令 等`
+ make - 让(驱使), let - 让(允许)
+ have - 给某人 任务/责任 做..., get - 说服/鼓励 某人做某事
  + I'll have the wolf wash the dishes. 我会让狼洗盘子
  + I had someone wash my car. - I had my car washed.
  + = My car was washed (by someone) as I wanted.

## 虚拟语气 - subjunctive

+ 用法一: 表示 不可能的 假设
+ 与现实相反的, 或是 主观意愿
  + 条件句: 过去式/were
  + 主句: would + 动词原形
+ 与将来实事相反(或几率非常小的事)
  + 条件句: should (shall) + 动词原形
  + 主句: would + 动词原形
+ 与过去实事相反(不可能发生的事)
  + 条件句: had + 过去分词
  + 主句: would have + 过去分词

+ 用法二: 表达愿望, 请求, 建议, 命令 等
+ 即希望 某个动作/事件 发生, wish / hope
+ 对现在的愿望: wish + 动词过去式/were
  + I __wish__ that I __were__ a rabbit.
+ 对过去的愿望: wish + had/would have + 过去分词
  + I __wish__ that I __had eaten__ the carrot.
+ 对将来的愿望: wish + should/would/could/might + 动词原形
  + I __wish__ that I __could eat__ the carrot.
+ 其他动词
  + __would rather__ 用法 = wish
    + I would rather that you hadn't eaten the carrot.
  + __demand, suggest, insist, order...__
    + 这几个后面接 __should + 动词原形__
    + I __order__ that you (should) give me a carrot.
      + 这里 should 可以省略
    + I __ordered__ that you (should) give me a carrot.
    + 宾语从句: I suggest that you (should) eat a carrot.
    + 主语从句: it is suggested that you (should) eat a carrot.
    + 主语补语/表语从句: The suggestion is that you (should) eat a carrot.
    + It is (high) time that you ate a carrot.

+ 如果我看到兔子, 我将要咬它
+ if I see the rabbit __now__, I __will__ bite him.
  + 在英语中不能这样表达, 因为 will 表示的是确信会发生的事
  + now 也是一个明确的时间, 都是在客观的描述实事

+ 将时间往前推一个时间段, 与现在实事相反, **动词变过去式**
+ if I __saw__ the rabbit now, I __would__ bite him.
  + 这里的过去式, 并非描述过去发生的事情
  + 而是一个主观意愿, 想象的假设

+ 表示与过去实事相反, 动词 使用过去完成时态
+ 如: 早知道如此, 当时我就该做那件事
+ If I __had seen__ the rabbit __a few days ago__, I __would have bitten__ him.

# 谓语动词

+ 实义动词
  
  + 有实际意义的动词,分及物动词和不及物动词
  + 分物种形式 : 原形, 过去式, 过去分词, 现在分词, 第三人称单数
    + 过去式经常和过去分词一样在结尾 + __ed__
    + 现在分词一般 + __ing__
    + 第三人称单数一般加 + __s__

+ 及物动词
  
  + 及物动词使用必须加宾语
  + 也就是说需要一个动作对象
  + 如: buy(买) ,买 _什么_

+ 不及物动词
  
  + 句型: 主语 + 不及物动词(谓语) + 状语
  + 意思完整的动词
  + 如: Work(工作)

---

+ 系动词
  + 不能够单独使用,需要和 表语 一起构成谓语
  + 最常用的系动词就是 be(是) , be 有八种不同的形式
    1. 原型 __be__ , 用于助动词和情态动词后面, 或者用在动词不定式里
    2. 第一人称 I 后面的 __am__ , 只有在一般现在时态中的 I 才会使用
    3. 第三人称单数 __is__
    4. You 和 复数后面的 __are__
    5. 单数的过去式格式 __was__
    6. 复数的过去式格式 __were__
    7. 过去分词格式 __been__ 完成时态
    8. 现在分词格式 __being__ 被动语态
  + 除了 be 还有一些 实义动词 也可以做系动词
  + 如: look(看) - look at(看起来) - look beautiful(看起来很漂亮)
  + 特点 : 原本都是实义动词 , 意思马上发生变化 , 必须跟形容词作表语
  + 特殊系动词 - 有系动词的作用
    + get, feel, look, grow, become, 
    + grow strong
    + get angry, feel angry

## 双宾动词

+ Give, send 第一类双宾动词 用 to 连接对象
  + I give him books = I give books to him
+ Buy, find 第二种 用 for 链接对象
  + I buy my daughters milk = I buy milk for my daughters

## 助动词

+ 没有意思,作用就是帮助 实义动词 或 系动词 构成 复合谓语
+ 常用的有 : be , do , have , will , would , shall , should
+ 特点: 没意思 , 不能单独使用,必须帮助构成其他结构 , 没有专门的助动词,都是从其他动词借过来的
+ 如 : I __am__ working now. 这里 am 就是借过来

## 情态动词 aux = auxiliary

+ 不能单独做 谓语 的 谓语动词, 必须跟 **动词原形**(实义动词 或 系动词 be)
+ 特点 :
  + 必须跟动词原形构成复合谓语
  + 有意思, 但是意思不完整
  + 一般不能跟助动词连用
  + 情态动词后面 只跟动词原型, 不需要任何其他格式
  + 含有情态动词的句子, 时态结构一般加上时间状语就行
  + 有些情态动词不需要跟时态变化 如: must
  + 有些需要变化 如: can - could(preterite)
    + may - might
    + have to - had to
    + be able to - was/were able to

+ 当想表达如 : 必须 , 能 , 应该 等等的时候就可以使用
  + I earn money every day.
  + I __must earn__(复合谓语) money every day.

+ can - could, may - might
+ Should, must, have to(不得不), dare(敢)
+ Needn't(不必), had better(最好)
+ Ought to = should, Be able to = can
+ used to, be about to

+ 如: can speak, can look after
  + must go, should find

+ 一般疑问句 - 情态动词移到开头
  + can she buy a house?
+ 否定句 - 情态动词后加 not
  + she cannot buy a house.
  + you can't smoke here.

+ 口语多用 can
+ May I use your phone?
+ May I come in?
+ May I help you? = can I help you?

+ might 可以不作为 may 的过去式使用, 而是表示更加婉转的语气
  + may/Might I go with you?
+ could 也有同样的用法

### must

+ 没有时态变化
+ 在肯定句中表示: **必须, 一定**
+ 疑问句
  + Must we finish the homework today?
  + Yes, you must.
  + No, you don't have to. answer 较为特殊
  + because "mustn't" express 意思不同
+ 否定格式: mustn't - 禁止, 不准
+ 主系表结构: 表示 猜想 - 准是, 一定
  + you must be hungry - 你肯定饿了
+ must have + 过去分词
  + 这个句型多翻译为: 以前/过去 一定做过某事

### have to / don't have to

+ 意思接近 must
+ 表示客观上的必要性, must 为主观必要
+ 如: I have to go to work on time
  + 我不得不去上班, 并不是我主观想去
+ 否定格式表示: 没必要
+ 三单格式用 has to / doesn't have to
+ 一般过去时 had to / didn't have to
+ 将来时态 will / won't(will not)  have to
+ 疑问句和否定句需要借助助动词 do 来构成
  + Do you have to leave soon?
  + You don't have to say sorry.

### ought to - 应该

+ 只有肯定句和否定句 - should 更常用
+ 语气比 should 强
+ 否定句: ought not to

### should - 应该

+ 否定: shouldn't
+ 疑问句: Should we help she?

### dare - 敢

+ 在肯定句中作实义动词
+ 疑问句, 否定句, 条件句中作情态动词
+ dare 及物动词, 没有人称变化
+ dare not / daren't 情态动词
  + I dare not eat durian - 对
  + 但是不能 : I dare eat durian - 因为 dare 在肯定句中是实义动词
  + I dare (to eat durian) - 对, 动词不定式短语做宾语

+ I (dare not/don't dare) look at her
+ 三种方式
  + don't dare
  + doesn't dare to
  + dared not

+ 只要在否定意思的句子中, 都可以变为情态动词
  + No one dared speak of it
+ 同样的在疑问句和条件句也会变成情态动词

### be able to - 能, 可以

+ 用法和 can 差不多
+ be/is/am/are (not) able to = can
+ was/were (not) able to = could
+ will (not) be able to = can (can 本身就带有将来时的意思)
+ 疑问句
+ you can type - can you type?
+ you are able to type - Are you able to type?

### needn't

+ 只有在否定格式 needn't 才是情态动词
+ needn't 动词原型 / needn't be
+ 与 dare 一样, 在否定意思的句子里也可以成为情态动词
  + I need __hardly(几乎不,带有否定意义)__ tell you the truth

### had better

+ 表示: 最好, 一般缩写为: 'd better
+ 否定格式: had better not - 最好不
+ You'd better take a nap after lunch
+ 固定写法, had 不能写成 have / has

# 非谓语动词

## 动词不定式 infinitive

+ 谓语动词和非谓语动词, 动词不定式就是 **非谓语动词** 的格式
+ 当一个动词不做谓语, 而是做主语, 表语, 宾语, 状语...,就叫非谓语动词
+ 谓语动词, 一般是 动词原型 或 过去式 等
  + I want __eat(不行, 谓语不能做宾语)__
+ 非谓语动词, 需要 增加符号 或 词尾字母组合 构成
  + I want __to eat (非谓语, 增加了符号 to)__
+ 虽然格式不一样, 作用不同, 但是意思是相同的

+ 动词不定式
  + 没有 **人称变化和时态干扰**, 在句中不能单独做谓语
  + 可以有自己的 宾语 和 状语, 一起构成 动词不定式短语

+ 基本形式
  + to + 动词原型, to 是符号没有意思
  + 特殊情况可以不带 to
+ 一般形式
+ 完成式
+ 进行式
+ 完成进行式

## 不定式短语

+ 作用 - 可以作: 主语, 表语, 宾语, 定语, 状语, 宾补
+ 表示否定意义, 在 to 前面加 not
  + To learn English - not to learn English
  + I decided not to learn English.

+ to 及物动词原型 宾语 (状语) - 表达 一个动作 / 一件事情
  + To buy a house
  + To answer a question
  + 状语需要注意区分是 不定式短语的状语, 还是主句的状语
  + To help us warmly
  + To meet her at the station

+ to be 表语 (状语) - 表达 一个状态 / 一个情况
  + be 是系动词, 本身也是一个谓语动词
  + To be happy - (是)快乐的 (be 不需要翻译)
  + To be here
  + 一般情况 形容词 和 介词短语 不能做主语
  + 但是通过 不定式的构成 都可以作为主语, 或其他
  + To be busy every Sunday

+ to 不及物动词 状语
  + To work every night
  + To swim in the river

+ to 动词短语的原型 (宾语) (状语)
  + To look after me every day
  + To get up early every morning

+ 疑问词 + 不定式短语 - 特殊的不定式短语, 用法相同
  + Who, what, which, when, where, how, whether
  + 这个特殊的不定式可以作 **介词宾语**
    + about to learn ... - 错的
    + about how to learn English
  + I don't know how to get there.

### 作主语

+ 做主语是单数概念
  + 主语 is/was 表语 - 逻辑很重要
  + To be a teacher is hard.
  + To woo her is a happy thing.
  + To work for the boss is impossible.
  + 但是这种方式头重脚轻, 英文喜欢用 it 代替不定式, 动词不定式放后面
  + it is impossible to work for the boss.

+ 不定式 想要表是**动作是谁做的**, 可以在 不定式 前加一个 for 短语
+ for + 名词(代词的宾格) + 不定式 , 这个作为不定式的**逻辑主语**
+ 增加逻辑主语的时候必须放后面
+ **it(形式主语) is/was 表语 (for/of sb) 不定式(真主语)**
  + it is necessary for me to learn English grammar.
  + 在这里 for 不翻译
    + 不能翻译成: 对我来说学英语语法是必要的
    + 而是翻译为: 我学英语语法是必要的
+ 如果这个句型的表语是由:
  + 某些**表示人品行的形容词** , 那么 for 要改成 of
  + 如: kind, right, rude, nice, clever ...
  + it is important for me to read English every day.
    + important 形容这件事很重要, 所以用 for
  + it is wise of me to read English every day.
    + wise 形容我做这件事很明智, 所以用 of

### 作表语

+ 不定式做表语, 主语不能是 **有生命** 的事物
+ 需要注意逻辑问题
+ 主语大多是一些抽象名词
  + Wish - 愿望, dream - 梦想, plan
  + mission - 使命, assignment - 任务
  + task - 任务, goal - 目标, ambition - 志向
  + aim - 目的 - purpose

+ Our goal is to finish our course in three months
+ Her wish is to become my wife.
+ My plan is to buy a very good computer afterward.

### 作宾语

+ 不是说有动词都可以直接跟 不定式作为宾语
+ want, need, like, wish
+ hate, prefer, hope, continue
+ manage, try, offer, start ......

+ I want to buy a house in the future.

### 作后置定语

+ 修饰一个名词, 要求比较严格
+ 名词
  + Way, reason, chance, opportunity
  + ability, effort, decision
  + plan, promise, wish

+ He was the first person __to think of the idea__.
+ The best way __to increase vocabularies__ is to read more books.

### 作状语

+ 通常可以做三种状语
  + 目的, 原因, 结果
  + 目的状语放开头需要加 逗号

+ 目的 - 说明谓语的目的, 做...是为了...
  + He does two jobs every day __to buy a big house__.
    + 做两份工作的 目的就是 为了买一个大房子, 要翻译出为了...
  + She learns English hard to pass the exam.
  + She came to My bedroom to sleep with me on yesterday night.

+ 原因 - 一般情况下只用在 主系表结构
  + 表语必须是 表示情绪的形容词
  + I am glad to hear from her.
    + hear from 固定搭配, 收到...来信
  + I was surprised to see him there.
  + I am lucky to marry her.
  + She was disappointed not to marry me.
  + 口语经常省略 it is / I am
    + Nice to meet you = it is nice to meet you

+ 结果
  + The Earth spins round once every day to give us day and night.
  + 地球每天自转一周给了我们日夜

### 宾补

+ 宾补用以补充 宾语的情况, 只有特定的及物动词才需要
+ 而且不同的 动词 搭配也不一样
+ ask 要求, tell 吩咐, invite, force
+ advise 建议, get 让, beg 乞求, allow 允许...

+ I ask her __to sleep beside me__.
+ My Mother told me __to come back early__.

+ 特殊情况, 跟的动词不定式 to 要省略, help 可以不省略
  + make, let, have, see, hear, find
  + watch, notice, feel, observe
  + listen to, look at, help

+ I make my students __check homework carefully__.
+ We saw a dog __bite a man__.
+ I often hear her __sing the song__.

## 动名词

+ 动名词与 不定式一样是 非谓语动词, 具有动词和名词的特征
+ 可以 **带宾语**, 受**状语修饰** , 一起构成动名词短语
+ 一般构成形式就是加 ing 与现在分词相同

+ 一般式
  + 主动: doing / being + 表语
  + 被动: being done
+ 完成式
  + 主动: having done
  + 被动: having been done

+ 动名词可以做很多句子成分
  + 主语, 表语, 宾语, 介词宾语, 定语, 状语, 宾补
  + 比 不定式 多了一个 介词宾语 的作用
  + 但是 疑问词 + 不定式 也可以做 介词宾语

+ 动名词 与 不定式 的区别: **主语, 表语**
  + 动名词: 表示 **比较抽象的一般行为时** 或 **多次的行为动作时**
    + 一般来说, 动作已经发生了, 常用的 job, hobby, habit
    + His job is selling houses. - 工作, 不可能是卖一次
    + smoking is not good for us. - 经常性的抽象行为
    + speaking in front of a lot of people os hard.
    + Buying tickets before spring festival is hard.
  + 不定式: 表示 **一次性的 / 动作**, 特别是 **将来的动作**
    + 动作可以是没有发生的
    + 也可以是已经或经常发生的动作, 但是强调是 **具体的一次动作**
    + it made me happy to find my cellphone. - 虽然已经发生了, 但是 找到手机 是一次性的
    + My plan is to but a house next year. - 动作还没发生, 且是一次性的动作
    + It is easy to find a job.
    + To buy a ticket is easy.

+ 否定格式
  + 加 not 否定 动名词的动作
  + I remember not giving him the book.

  + being + 表语

+ trying to help her to learn English.

### 作主语

+ 属于单数概念
  + 主要用于 主系表 - is / was
  + Learning new words is very useful to me.
  + Reading English loudly is necessary.
  + 做主语不需要像 不定式一样 移动到句尾
  + 只有一些特殊的表语, 才需要移动到句尾
    + no use, no good, fun, dangerous
    + Nice, a bore, useless, a waste of time
    + tiring, a good pleasure, senseless, enjoyable
  + it is useless arguing with SB.

+ 特殊 - 不能把 动名词 移到后面
  + essential
  + important
  + necessary

+ 主谓宾
  + Drinking milk helps bones __to grow strong(宾补)__

### 表语

+ 作表语 与 不定式相同

+ My favorite sport is swimming.

### 作宾语

+ 必须是特定的 及物动词 , 与 动词不定式的跟的及物动词不是同一类

+ can't help, mind, enjoy, admit
+ require, postpone, delay, consider
+ deny, advise, excuse, despise, like
+ practice(se 是英式动词, ce 做名词, 美式动词名词都用 ce)
+ escape, miss, pardon, imagine, risk
+ put off, leave off, appreciate, keep
+ insist on, give up, feel like

+ I enjoy listening to music.
+ Did you finish reading the book?
+ I couldn't help laughing.
  + can't help 跟*动名词*才能翻译为 忍不住

### 作介词宾语

+ 特定的介词才可以
  + about, after, before, without
  + by, of, instead of, for ......
+ 带有介词的动词短语, 后面可以接 动名词短语 作 介词宾语
  + give up, think of, look forward to
  + dream of, hear of, depend on
  + insist on, feel like, succeed in
  + complain about, stop sb/sth from doing sth

+ be good at - 擅长于...
  + I am good at swimming.
+ be interested in - 对...感兴趣
  + She is interested in singing songs.
+ be proud of - 为...感到自豪
  + She is proud of loving me
+ be afraid of - 对...感到害怕
+ be fond of - 喜欢
+ 这三个 in 要省略, 后面的动名词不变
  + spend ... in - 花...做...
    + I spend a lot of time Learning English.
  + be busy in - 忙于
  + have trouble/difficulty in - 做...有困难

+ about - 有关, 关于
  + 跟动名词主要做 表语, 后置定语
  + about Learning English
  + about having a party

+ after / before - ...之后/前
  + 主要作 时间状语, 可以放开头或句尾
  + After getting up every day, I turn on my computer.
  + After watching the video, she was excited.
  + Before sleeping every night, she kiss me.

+ without - 没有做...就...
  + she sleeping without eating dinner last night.
  + She left my home without saying goodbye.

+ By - 通过...动作
  + I learn English by watching video.
  + She earns money by selling flowers.

+ of - ...的
  + 后置定语 修饰 抽象名词
  + My way of learning English is different.
  + it can reduce the risk of infecting the disease.

+ Instead of - 没有...而...
  + 没有做什么事情, 而是去做另外一件事, 强调做出 other choice
  + Instead of telling me the truth, he told me another thing.
  + Instead of helping the old man stand, he called the police.
  + Now I can walk to work instead of going by car. 
    + 现在我可以步行去上班，而不必开车了。

+ for - 原因状语
  + 大多数出现在被动句中, 或固定句型
  + He was arrested for killing a girl.
  + Thank you for helping me.

## 不定式与动名词作宾语的区别

+ like 和 hate , dislike 和 love
  + 可以跟 不定式 和 动名词
  + like doing sth - 表示爱好
  + like to do sth - 想做某事
  + hate doing sth - 讨厌做某事
  + hate to do sth - 不想做某事, 强调某一次, 而不是一直不想, 也可以表达不得不做
+ begin, start, continue 这三个区别不大

+ remember
  + 不定式: 记住要去做某事
  + 动名词: 记得曾经做过某事
+ forget
  + 不定式: 忘记去做某事
  + 动名词: 忘记曾经做过某事
+ regret
  + 不定式: 对要做或正在做的事情抱歉
  + 动名词: 对已经做过的事情懊悔
+ stop
  + 不定式: 停下来去做另一件事
  + 动名词: 停止做某事
+ try
  + 不定式: 试图做某事
  + 动名词: 尝试做某事
+ mean
  + 不定式: 打算做某事
  + 动名词: 意味着, 意思是
+ Go on
  + 不定式: 做完一件事, 接着做另一件事
  + 动名词: 重复做相同的事情

+ want / need / require
  + 动名词: 需要 和 need 同义, 用在被动语态
    + His computer needs/wants repairing
  + 不定式: , 主动语态
    + He needs to repair his computer.

## 现在分词

+ 与动名词的作用不一样
+ She is charming
+ The story is interesting
+ The game is exciting

+ 主要区别
  + 动名词: 在句中的作用就是名词, 可以与主语互换
  + 不包含 进行 的意思
  + 修饰名词时表示用途: a swimming suit - 泳衣
    + My hobby is swimming.
    + Swimming is my hobby.
  + 现在分词: 在句中的作用是说明主语性质, 不能与主语互换
  + 可以含有正在进行动作的意思
  + 修饰名词的事件性质或状态: a swimming boy - 正在游泳的男孩
    + The story is interesting.
    + Interesting is the story. - 这明显不对

+ 动名词: a sleeping pill = a pill for __sleeping__
  + 这里的 sleeping 作用是名词
+ 现在分词: a sleeping rabbit != a rabbit for sleeping
  + 这里的 sleeping 作用是形容词

+ 作宾补时
  + 不定式: 强调动作已经发生了
  + 现在分词: 强调动作正在进行中

## 过去分词

+ 构成 - 动词过去式
+ 作用: 定语(可看做形容词), 表语(被动), 状语
+ 过去分词短语作定语, 要放到被修饰的名词之后 (后置定语)
  + The carrot cake eaten by the rabbit is delicious.

+ 使用 现在分词(主动) 还是 过去分词(被动), 主要是分清 主动 和 被动

+ 表语 - rabbit 被某物引起了兴趣 - 有被动含义
  + The rabbit is interested in growing carrots.
  + 这兔子被...吸引了 - 这兔子对种萝卜感兴趣

+ The story is __interesting(主动)__.
  + 这故事让...感兴趣 - 这故事是有趣的
+ frighten - 使害怕
  + The rabbit was frightened. - 被动
    + 兔子被...吓到了 - 兔子很害怕
  + The rabbit was frightening. - 主动
    + 兔子使...害怕 - 这兔子很可怕

+ 状语 - 地点, 原因, 条件
  + Seen from the hill, the carrot field looks beautiful.
  + Born in a rabbit family, the rabbit has been growing carrots all his life.
  + Given another chance, the rabbit would go to the wolf's party.

## 独立主格

+ 通过 现在分词 和 过去分词 省略从句

+ 从句: Because he was invited by the wolf, the rabbit decided to go to the party.
+ 过去分词: Invited by the wolf, the rabbit decided to go to the party.
  + 通过过去分词简化, 只拥有一个谓语

+ 从句: Because the wolf invited him, ~.
+ 独立主格: The wolf (独立主格) inviting (现在分词) him, ~

+ 从句: If the weather permits, ~.
+ 独立主格: The weather permitting, ~

+ After the work was finished, ~
+ The work finished, ~.

+ The rabbit

# 冠词 art = articles

+ 冠词是一种虚词, 没有词义(中文可能有意思), 不能单独使用

+ 不定冠词 如: _a , an_ 意思相同 音标元音
  
  + 只能跟可数名词单数
  + a 用在辅音开头的名词前
  + an 用在元音开头的名词前
  + 在指一类人或事物等泛指的情况不用翻译

+ 定冠词 如: _the_
  
  1. 在普通单数名词前指一类人或物 不翻译
  2. 在重新提及的人或事物前 含有 **这,这些** 的意思
     + I bought a book yesterday. The book is interesting
  3. 在谈话双方都知道的 人或物, 可以修饰单数,复数,和不可数名词
  4. 在有限定性后置定语修饰的名词前
     + the man in red - 穿红衣服的人
  5. 凡是特指的名词都要加 the
     + 如描述指定的一本书就可以使用 the book
  6. 在序数词前,或世界中独一无二的名词前
  7. 在表示时间或其他情况的习惯用语前
  8. 在形容词和副词的最高级前
  9. 在某些形容词前,变成名词
  10. 在姓氏复数前,表示一家人或夫妇

+ 不使用冠词
  
  + 专有名词(人名、地名、国名、节日、月份、星期、季节)
  + 称呼语
  + 表示职位、身份、头衔的名词
  + 物质名词或抽象名词
  + 可数名词复数或不可数名词表示泛指的时候
  + 表示科学、疾病、球类、游戏、颜色、感官名称
  + 与 by 连用的表示交通工具的名词前
  + 在名词前已经有形容词性物主代词,名词所有格

+ 首次提到某人某物,并非特指
  
  + He is a doctor
  + I am a teacher

# 时态

+ 用来确定句子中发生的事是在什么事件点
  + I __bought__ a book _yesterday_
  + I __will buy__ a book _tomorrow_
  + I __buy__ a book _every day_
+ 根据时间点的不同, 谓语动词要发生对应的变化

## 一般现在时

+ 使用一般现在时的情况
  + 经常性,习惯新,有规律性的动作或状态
  + 表示真理性,客观性的动作和状态
    + The earth is round 地球是圆的
  + 不会轻易的改变状态 (主谓宾)
  + 此时此刻的状态 (主系表)
  + today 只能用在主系表结构中,因为主谓宾表示的是不会轻易改变的状态
  + 解说 or 描述: 情节,比赛... 都用一般现在时

+ 实事陈述
  + I eat rice. - rice 不可数, 表示种类
+ 表达习惯/重复动作
+ 标记预计发生的事 - 相当于陈述实事

### 主系表

+ 主语决定 **系动词**

+ 否定句
  
  + 变否定句 在 系动词后加 not
  + I am not
  + rich is not important food

+ 疑问句
  
  + 把 系动词 移动到句首,根据人称改变发生变化
  + are you a teacher?
  + is she pretty? - 肯定句: She is pretty

+ **人称代词(主语)** + is/are/am + 形容词
  
  + _I_ 跟 __am__
  + 单数名词主语用 __is__
  + _复数和 you_ 用 __are__
  + I am / you are / she is / he is / it is / they are / we are

+ **名词(主语)** + is/are + 形容词
  
  + 专有名词里的人名和地名作主语时,一个人或一个地名主语用 __is__
  + 可数名词单数用 __is__
  + 可数名词复数用 __are__
  + 不可数名词,属于单数概念用 __is__ 不可数名词作主语也强调泛指,如果是特指,依然需要考虑加**定冠词 the 或限定词**
    + water is important - 水是重要的 - 泛指所有水
    + the water is important - 这水很重要 - 限定了范围

+ 人称代词主格(主语) is/are/am + **名词**
  
  + **人名**, 用人名做表语不需要任何变化,直接放在表语的位置就行
  + 表示职业身份的名词,这种多数是可数名词,需要注意加**限定词,冠词或变复数**

+ 名词 is/are 名词
  
  + 两个名词都需要注意限定词和复数的问题

### 主谓宾 (主谓,双宾,宾补)

+ 主语第一二人称单数和复数(I,we,you), 第三人称复数(they)以及名词的复数的时候 **谓语动词使用原型** *主语和宾语不用发生变化*
  
  + *非第三人称单数*

+ 主谓
  
  + I sleep(不及物动词) every day 我每天睡觉

+ 名词复数(主语) + 不及物动词原型(谓语) + 状语
  
  + these(限定词可以让句子更加合适) teachers work every day
  + I and you work every day

+ 人称代词主格(主语) + 及物动词原型(谓语) + 宾语 + 状语
  
  + I bake beefs every day

+ 名词复数(主语) + 及物动词原型(谓语) + 宾语 + 状语
  
  + The men Cook noodles every day

+ 疑问句
  
  + My friends work every day
  + Do My friends work every day?

+ 否定句
  
  + I read books every day
  + I don't read books every day

### 第三人称单数 (三单)

+ 第三人称单数时,**谓语动词**要变成**第三人称单数的格式**

+ 除开 '我'和'你' 的其他第三者(人或物)单数, 且是主语

+ 第三人称单数的词语
  
  + 人称代词主格 : she, he, it
  + 所有的名词单数: bus, father, doctor
  + 一个人名,一个地名
  + 不可数名词: water, sugar, wine
  + 指示代词: this, that
  + 动词不定式和动名词短语
  + 主语从句

+ 句型(谓语都用三单格式)
  
  + (she / he / it) + 谓语
    + he works every day
  + 名词单数(三单) + 不及物动词 + 状语
    + 名词单数不能单独使用, 需要加限定词
  + 人称代词主格(三单) + 及物动词 + 宾语
  + 名词单数(三单) + 及物动词 + 宾语

+ 疑问句
  
  + **句首 + (助动词)does(无意思)**
  + **谓语变回原型**
  + She likes playing game
  + __Does__ she __like__ playing game?
  + 肯定回答 : yes, she does
  + 否定回答 : no, she doesn't

+ 否定句
  
  + **谓语动词前加 does 的否定式 doesn't**
  + **谓语变回原型**
  + he has a cat
  + he __doesn't__ __have__ a cat

## 现在完成时

+ have 为助动词时, 没有意思
+ 动词的过去分词不能单独使用, 必须跟跟助动词 have

+ 第三人称单数助动词用 has

+ 主谓宾

  + She has baked potatoes
  + He friends have drunk beer

+ 主系表
  + have/has + been

  + 主动移到开头, 动词过去式 **不变**

+ 否定句
  + I have been busy - I have not been busy

### 用法一

+ 一个 **动作或状态** 一直延续到现在
+ 谓语动词必须是 **延续性动词**
+ 如 : die 就是瞬间性动词, 因为 死 不能一直死
+ 如 : learn 就可以延续, 因为可以一直学习
+ I have learned English for five months
+ he's been dead for fifty years
  + 状态是可以延续的, 但 死 这个动作不行

+ 表示时间延续的状语
  + for two years - 两年了
  + since 1998 - 自从1998到现在
    + since 要跟 已经过去的准确时间点
    + since last week , since two days ago
    + since + 一个一般过去时的句子, 属于状语从句
  + all day - 一整天

### 用法二

+ 表示过去发生的 某一动作 对 现在造成影响和结果 的时候
+ 动词要求是 **非延续动词** , 如 die , buy
+ 这种用法只能用 **不确定** 的时间状语
  + 如 already(已经) - yet(仍然) - once(一次)
  + twice(两次) - just(刚刚) - never(从不)
  + before(之前) , already/just/never 放在 助动词 后面
+ 强调 对现在的 __effect__
+ 在否定形式的情况下可以跟 延续性的时间状语 for, since
  + I haven't seen you for ten years.
  + 在否定句中意思发生了改变 - 多久没有做某事了

+ I have already known the story
  + already 加不加意思都一样, 因为 该时态本身就有 已经 的意思
+ He has died - 他 **已经** 死 **了**
+ He has __gone to__ somewhere
  + 强调已经出发去了某地, 人不在这里
  + 不能描述 自己 已经去了某地
+ He has been to somewhere
  + 强调曾经去过某地, 但是现在已经不在该地

### 与一般过去时 的区别

+ 第二种用法的三个特点
  + 强调动作已经发生过了, 但是不知道什么时候发生
  + 强调对现在产生的影响, 不强调过去
  + 不需要具体的时间状语, 也不能加过去和现在的时间状语
+ 如 : Have you finished your homework?
  + 重点是作业是否做完了, 而不关心什么时候完成的

+ I bought the book
  + 强调过去买过
+ I have bought the book
  + 强调过去买过, 已经不需要买了

# 副词 adv = adverbs

+ 副词是修饰 动词, 形容词, 副词 的词语
  + 有时也可修饰 数词, 介词, 连词, 名词 或 全句
+ 句子成分: 状语, 定语

+ 时间,地点,方式副词主要做状语

+ 程度副词用来修饰形容词和方式副词

+ 频度副词也做状语,但位置不一样

+ 副词做状语书写顺序

+ 方式 - 地点 - 时间
  
  + I fish happily __here__ __everyday__

## 时间

+ 不同的时间副词使用在不同的时态中
  + 一般位于句尾, 有时会为了表示强调放在 句首
  + 也有些会放动词前, 如: already

+ now - 此时此刻 - 一般用在**一般现在进行时中**
  + 特殊情况特殊对待

+ soon - 很快 - 主要用在**一般将来时态**
+ every day - 每天 - 用在**一般现在时**,也可用在将来或过去时态
+ today - 今天 - 主要用在**一般过去时和一般将来时**
  + 一般现在时态只能用主系表结构

+ tonight - 今晚 - 用法如今天
+ tomorrow - 明天 - 用于一般将来时态
+ yesterday - 昨天 - 用于一般过去时态
+ early(早期的) 和 late(晚期的) - 可以直接用在一般现在,过去,将来时态
  + 且后面可以加一个时间状语
  + he goes to school early
  + I get up (early/late) every morning 早上起得(很早/很晚)

+ just now - 刚才 - 一般过去时
+ 后置定语 - yesterday, today, tomorrow ,tonight
  + the meeting today is very important. 今天的会以很重要
  + the lesson tomorrow is easy. 明天的课很容易
  + I don't know the meeting yesterday 我不知道昨天的会议
    + 这里的 yesterday 表示后置定语, 所以不用过去时

+ this/that + 时间名词单数
  + this 用法如 today - that 用在一般过去时态
  + this morning 今天早晨
  + that morning 那天早晨

## 地点

+ 主要做地点状语
  + 一般位于句尾, 有时会为了表示强调放在 句首

+ 后置定语
  
  + those buildings __around__ are very high 周围的建筑
  + the woman __upstairs__ is sexy 楼上的女人
  + we often go to the store __downstairs__ 楼下的商店

+ 做表语
  
  + I am __here__. 我在这里
  + she will be __here__. 她会来这里

+ 宾补
  
  + 能用地点状语做宾补常用的动词有: let,leave,keep,put
  + let him __out__ 让他出去
  + Put it __down__ 把它放下
  + I put my money __here__

## 方式

+ 表示**动作**和**心理活动**的词语成为方式副词

+ 用在主谓宾结构做方式状语

+ 大多数方式副词由 - **形容词 + ly** - 构成

+ 方式副词只能放在动作的句子后做方式状语
  + 有些也可放在句首 或 居中
  + I love you __very much__ 我非常爱你
  + I eat apples __happily__ 我开心的吃苹果
  + She cried __suddenly__ 她突然哭了
  + She cried __loudly__ 她大声哭

+ 多个方式状语 用 and 并列书写
  
  + suddenly and loudly

## 程度

+ 如: 很, 非常, 特别, 好 等
  
  + almost 几乎, nearly 几乎, very 非常
  + Rather 相当, fairly 相当, so 如此
  + much ...得多, quite 十分, pretty 很
  + extremely 非常, too 太, super 超
  + really 真正地, completely 完全地
  + extraordinarily 格外地, as 同样

+ 程度副词相当于一种限定词的作用
  
  + 主要修饰形容词和方式副词
  + 一般放在**形容词或方式副词**的前面
  + very good 非常好
  + so good 很好
  + too bad 太差
  + pretty good 很好
  + He is running __very quickly__ 他跑德非常快

+ much 作为程度副词
  
  + 必须修饰**形容词或方式副词**的比较级
  + much better 好得多
  + much bigger 大得多

+ 个别程度副词还可以修饰动词或数词,甚至名词
  
  + He __nearly__ died four years ago. 四年前他差点死了
  + The man __almost__ dropped his gun

+ really 的位置和频度副词相同
  
  + I am really tired 我真的很累
  + I really love her

## 频度

+ always 总是, often 经常, usually 通常
+ sometimes 有时候, seldom 很少, just 刚刚
+ Occasionally 偶尔, never 从不

+ 频度副词也做状语, 但是位置不一样

+ **频度副词放实义动词前**
+ be 动词/助动词/情态动词 之后
  + I __often__ eat rice 我经常吃米饭
  + I _will_ __always__ _love_ you

+ 主谓宾 - 修饰谓语
  
  + We __never__ swim
  + We don't __often__ swim

+ 主系表 - 修饰形容词
  
  + I am __always__ busy 我总是很忙
  + I am __often__ late 我经常迟到

## 特殊

+ high 高的, deep 深的, hard 艰难的
+ Early 早的, fast 快的, pretty 漂亮的
+ enough 足够的 ......

## 其他副词

+ 疑问副词
  + How 对方式提问
  + Where 对地点提问
  + When 对时间提问
  + Why 对原因提问

+ 链接副词和关系副词 引导从句
+ 句子副词
  + 一般放开头, 用逗号隔开

## 比较级和最高级

+ 大多只有方式副词才有比较级和最高级

+ fast - faster - fastest

+ hard - harder - hardest

+ quickly - more quickly - most quickly

+ 方式副词只能做状语,修饰动作,放在动作的句子后
  
  + _than B_ 比较状语
  + A + 动词 + 宾语 + 方式副词比较级 + than + B
  + I run __faster__ __than(比)__ _he(和谁比)_
  + He speaks more slowly than me

+ A + 动词 + 宾语 + 方式副词最高级 + 表示范围的词语
  
  + 加范围只是更加符合逻辑, 否则就是泛指
  + Who runs fastest in your class?

# 介词 prep = prepositions

+ 介词不能单独使用
  
  + 一般只能用在 名词 、代词 、 数词 、 动词 的**前面**
  + 介词跟这些词语一起构成一个短语
  + 主要用做：**状语、后置定语、表语**、*宾补、主补*

+ 如：
  
  + in HuBei 介词 + 名词
  + in 2000 介词 + 数词 - in + 表示年份的数词,表示在某一年

+ 常用的介词
  
  + In/of 、at 、on 、with 、for 、from of
  + as 、below 、between 、by 、during
  + for 、from 、inside 、like 、near
  + off 、past 、around 、to 、since
  + through 、up 、without 、against
  + among 、across 、out of 、into
  + along 、outside 、after 、about
  + Under 、behind 、beside 、before

+ 介词后跟的词语叫 - 介词宾语
  
  + 名词 、代词 、 数词 、 动词, 可以作为介词宾语 
  + 人称代词用宾格

---

+ 介词 + 动名词
  
  + before having dinner - 吃饭前
  + about learning English - 关于学英语
  + By watching TV - 通过看电视
  + 这里并非现在进行时,而是组合成为介词短语方式

+ 动词和介词构成的动词短语也叫复合动词, 属于实义动词的范畴 - phrasal verbs - 短语动词
  
  + 动词短语意思大多固定
  + look at - 看 - 及物
  + get up - 起床 - 不及物
  + look for - 寻找 - 及物
  + look after - 照顾 - 及物

+ 介词分类

+ 简单介词 如：in 、after

+ 符合介词 如：inside 、into

+ 分词介词 如：including 、following

+ 短语介词 如：in front of

## 状语

+ 可以构成地点状语的介词 in, on, under, beside ...

+ 只要可以表示**地点、位置或方位**都可以做地点状语

+ 一般放句尾，特殊情况放句首或居中

+ 表示事件概念时间状语 in, at, after, during ...

+ 时间状语需要注意时态的变化

## 后置定语

+ 表示**地点位置**的介词短语,基本上都可以放置一个**普通名词**后面作后置定语

+ 后置定语只能修饰**普通名词**, 也就是可数名词和不可数名词,一般不能修饰专有名词

+ 表示时间和方式的介词短语一般不能当后置定语

+ The mountains __in HuBei__ - 湖北的山
  
  + 作后置定语修饰 mountains

+ The table under the tree - 这棵树下面的桌子

## 表语

+ 做表语直接放在主系表中助动词后面就行
+ 只需要注意时态
+ I am under the tree
+ I am in the room

## 补语

+ 只有少数及物动词后面可以跟介词短语做宾补

+ 常见的有 put , keep , place

+ I put my money in my drawer - 我把我的钱放在我的抽屉里

+ 做主语不足语 - 一般在被动语态中, 一般语态就是宾补
  
  + My coat was put on the bed by me - 我的大衣被我放在床上

## 基本规则

+ 凡是表示**地点,地方,位置**的介词短语都可以做**地点状语,后置定语和表语**
+ 做地点状语, 一般不需要考虑时态问题

## 用法多

### in

   + 还有一些固定的词语
   + in class 在课堂上
   + in bed 在床上,躺在床上,卧病在床

3. 表时间(大范围) , 在某年 , 某月 , 季节 , 世纪
   
   + In 2000 - 在 2000 年
     + 不需要加 years, 加 years 表示多少年后
   + In May - 在五月份
   + In winter - 在冬天
   + in the twentieth/20th century - 在20世纪
   + 时态看语境

4. 在...之后 - 表示事件长度的词语
   
   + a/an/one/数词 + 表示时间的单词
   + 做时间状语,基本用在一般将来时
   + In three days - 三天之后
   + In a week - 一个星期后

---

5. In + 表颜色的词语/衣服的词语 = 穿什么颜色的衣服
   
   + 表示颜色的词语不需要加冠词,也不能变复数
   + 基本只能做表语和后置定语, 一般不做状语
   + In red - 穿红色的衣服
   + the man in yellow is cool 
+ In + 表示状态的词语 = 处于...状态
  
  + his life is in danger - 他的声明处于危险当中
  + you are in trouble - 你有麻烦(处在麻烦当中)

6. In + 语言/语调/笔墨/颜色 = 用...方式/语言
   
   + 作方式状语
   + In chinese - 用中文
   + In English - 用英语
   + It is written in ink - 这是用墨水写的
   + in + 表示语气的词语
     + in a low voice - 低声说

7. In + 表示范围,领域,关系的名词 = 在...方面
   + 基本做状语
   + that country is rich in natural resources
   + in progress - 在进行(状态)中
   + in a dilemma - 处于进退两难的境地中

8. 表示运动或动作的方向,进,入,朝
   
   + 这种用法主要跟人特殊的实义动词搭配,有时作宾补
   + he put a pen in his bag - 他把一只笔放进他的包里
   + 表示这些意思时有时属于动词范围,如:
     + come in - get in - go in

### at

1. 表示地点 - 在...地方, 在...附近
   
   + at + 表示小地方的名词(建筑物或专有名词)
   + at the hotel - 在酒店
   + at the cinema - 在电影院
   + in 强调在什么里面(强调内部空间)
   + at 强调在什么位置(不强调内部空间)
     + at the bus stop - 在公共汽车站牌
     + at the door - 在门口

2. 表示时间(准确的时间点) - 在...时刻
   
   + at + 表示事件概念的词
   + at 19:59 - 在 19点 59 分
   + at 6:00 o'clock - 在 6 点整
     + o'clock = of the clock = ...点钟
   + at the age of 21, 表示在多少岁的时候

3. 表示状态或从事于 - 在..., 正在...
   
   + 基本为固定的介词短语
   + at table - 在进餐
   + at war - 在打仗
   + at work - 在工作,在运作
   + at school - 在上课,在求学
   + at breakfast - 在吃早餐
   + at play - 在玩
   + at rest - 在休息
   + at peace - 处于和平状态

4. 表示速度,价格 - 以..., 按...
   
   + I bought it __at a low price__ - 我以低价收购了它
   + the pen was __sold(被动语态)__ at three dollars - 这支笔卖3美元
   + He drove __at 60 miles an hour__ - 他以每小时60英里的速度开车

5. 表示动作方向 - 朝,向
   
   + 动词 + at = 动词短语, 也是固定的
   + don't laugh at him - 不要嘲笑他
   + look at - 看
   + aim at - 瞄准
   + smile at - 对...微笑
   + shoot at - 向...射击

6. 习惯用语
   
   + be good at - 擅长于...
   + be bad at - 不擅长...
   + be expert at - 精通于... = be very good at
   + 主系表 + 固定
+ at + 集体性活动 - 构成一个介词短语
  + at party
  + at wedding
  + at meeting
  + at funeral

### on

+ Dinner is on me. - 晚餐我付钱 - 为晚餐付钱的重任在我身上
  + The burden of paying for dinner is on me.

1. 位置 - 在...上面
   
   + on + 表示物体的名词或代词
   + 强调在这个物体的上面,并且必须强调跟物体的表面接触
   + 做地点状语, 后置定语, 表语
   + on the roof - 在房顶
   + on the floor - 在地板上
   + on the wall - 在墙壁上
   + on the keyboard - 在键盘上
   + 写...在上面 用 in, 不能用 on

+ 也可以表示靠近某位置
  + at or near a place 在，接近（某地）
  + a town on the coast 沿海的城镇

2. 时间(小范围) - 在...时候
   
   + on + 表时间的名词 , 一般是指**一天内**的时间
     + 在**某一日**, 星期几, 日期 或某日的 上午, 下午...
   + on Monday - 在星期一
   + on Tuesday - 在星期二
   + on Wednesday - 在星期三
   + on Thursday - 在星期四
   + on Friday - 在星期五
   + on Saturday - 在星期六
   + on Sunday - 在星期天
   + 日期 - 月日年, 日月年 - 也可以不要年
     + on May 1st - 在五月一
     + on July 15 - 在七月十五号
     + on July 15, 2022 - 在2022年七月十五号
   + 在某一天的 morning, noon, afternoon, evening
     + on Sunday morning - 在星期天的早上
   + 具体到某一天的
     + On the ... of ...
     + on the night of May 5 - 在五月五号的夜晚

3. 表示状态 - 处于...状态
   + 基本为固定介词短语
   + on duty - 值班,之日
   + on fire - 着火
   + on a diet - 在节食
   + on drugs - 吃药

4. 表示关系 - 用法与 about 相同, 正式用语

5. 对什么敌方宣战, 用 on
   + He declared war on the little colony

### by

+ 含有 by 的动词短语
  + stand by 支持
  + pass by 路过
+ 被动语态
  + by sb/sth 被某人/某物

1. (时间)不迟于, 到...时为止
   + 用在现在完成时态 或 过去完成时态
   + he ought to be here by now - 到现在为此,他应该到这里了
   + by the end of this term, I had learned 300 words.

2. (位置)靠近, 在...旁边, 在...身边
   + 用法和 beside 相同
   + by 可以强调靠近, beside 只是在旁边

3. (方法, 手段)靠, 用, 以, 通过, 乘
   + by + 表示交通工具的名词/动名词/名词
     + 不需要加冠词或限定词也不用变复数 - 防止和第二种用法弄混
     + 用作方式状语
     + by bike - 骑自行车
     + by subway - 坐地铁
     + by taxi - 打的
     + by air = by plane - 坐飞机
     + by sea = by ship - 乘船
     + 主语 + go to + 地点/位置 + by
     + 主语 + travel to
   + by + 动名词 - 通过什么动作, 什么方式
     + I increased my words by reading more novels

4. (运动方向)沿着, 经过, 经由
   + 多数情况下跟特定的动词搭配
     + go, walk, run, swim, fly
   + I go to school by his house - 我每天路过他家去上学
   + I run by the park every morning

5. (标准, 度量, 数量)根据, 按...以...
   + don't judge a man by his clothes/look - 不要以貌取人

### for

+ 动词短语
  + look for - 寻找
  + ask for - 要求
  + be good for - 对...有好处
  + be famous for - 以...出名

1. (表示目的, 目标, 用途)为了..., 替代..., 对于..., 给...
   + for + 名词/代词
     + for + 表示人或抽象概念的名词
     + for + 代词宾格
     + 为了...
     + 作 目的状语 和 表语
     + for beautiful girls - 为了漂亮的女孩
     + for My wife - 为了我的妻子
     + for power - 为了权力
   + for 延伸出 帮/给
     + I will buy a book for you - 我会(帮/给)你买一本书
     + 双宾 I will but you a book
   + 表示 给... 的时候, 一般作表语
     + the letter is for you - 这封信是给你的
   + 表示 对于..., 一般为固定结构
     + tea is good for your health - 茶对你的健康有好处

2. (表示方向, 倾向, 能力)向, 往
   + they went out for lunch - 他们出去吃午饭了

3. (表示时间, 距离, 数量)经历, 计, 达
   + for + 冠词/数词 + 表示时间的名词 - 表示时间延续了多久
   + 一般用在现在完成时态
   + for five days - 五天了

4. (表示交换, 补偿)与...交换, 用
   + 主要用于固定搭配
   + exchange A for B - 拿 A 来换取 B

5. 支持, 赞成
   + for + 表示计划/方案 - 很少用
   + are you for this plan? - 你支持这个计划吗?

6. 因为, 由于
   + for + 动名词短语
   + 作原因状语, 多数用在被动语态

7. 当做, 作为
   + 用法 如 as
   + 主语 + have/eat + 表示食物的名词 + for/as + 餐名
   + I eat rice for lunch - 我中餐吃米饭

8. 至于, 关于, 就...而言
   + is's too hot for April - 就四月份而言, 天太热了

### from

1. (时间和地点的起点)从, 自...
   + 从一个时间点 到 另一个时间点 需要与 to 或 till 连用
   + from + 时间 to/till + 另一个时间
     + 时间点: 几点钟, 月份, 年份, 年龄, 星期几, 跟 after 差不多
     + from 20:00 to/till 22:00
     + I play the game about fps from 18:00 to 22:00
   + from + 地点 + to + 另一个地点 - 从一个地点 到 另一个地点
     + I run from My house to the school
   + from + 地点/位置的词语 and  from + 地点副词
     + she just came back from the school

2. (表示距离, 间隔)离
   + 一般出现在主系表 or 固定结构
   + how far is your hometown from here? - 你家乡离这里多有远
   + HuBei is very far from Beijing - 湖北里北京很远
   + 主语 be far/close from/to ...
     + far 远用 from
     + close 近用 to

+ 表示 **离...** 时的常用句型
  + 主语(表示地点的名词) is 数词 度量单位 from 另一个地点
  + My home is 200 meters from the river

3. (表示来源和出处)从..., 由..., 来自...
   + 作 后置定语 和 表语
   + I am from DaYe
   + where are you from?

4. (表示原因, 动机, 理由)出于, 由于
   + 一般为固定搭配
   + he died from an accident - 他死于一场车祸
   + die from - 死于非疾病
   + die of - 死于疾病
     + She died of cancer

5. (区别)...和...
   + 固定句型
   + 主语 tell A from B - 区别 A 和 B
   + can you tell a wolf from a dog? 你能区别狼和狗吗?
   + 

6. (表示原料, 材料)用...制成
   + 固定句型, 被动语态
   + be made from - 由...所做成的
   + the glass is made from sand

+ 固定短语
  + from door to door - 挨家挨户
  + from place to place - 到处

### about

+ 关于, 大概
  
  + 表示关于时 + 名词 、代词（宾格） 、动名词短语 、动词不定式
  + 表示大概是 + 数词

1. 关于, 有关
   
   + 做后置定语和表语, 一般不作状语
   + about TW(人) - 关于 TW
   + About Hubei - 关于湖北
   + about My book - 关于我的书
   + 做后置定语, 需要注意逻辑问题, 不能乱修饰
   + 作表语也需要注意逻辑问题
   + this book about me - 这本书是关于我的
   + this news is about her
   + the movie is about my dog

2. 在...附近
   
   + 用法跟 around, near 差不多, 可以做状语
   + 一般与固定的动词搭配

3. 在身边
   
   + 在身边 , 一般为固定结构: have + sth + about + sb(跟主语是同位语关系)
   + I have a cellphone about me - 我随身带着一部手机

4. 大概,大约
   
   + about + 数词, about 可以修饰数词
   + 跟几点钟的数词 - 做时间状语
     + about 3:00 p.m - 大概下午三点
   + I have 20 students - 我有20个学生
   + I have about 20 students - 我有大概20个学生
   + about 40 people attend my class every time - 每次大概有 40 人参加我的课程

+ 固定搭配
  + be about to - 将要, 如 be going to 一样
    + 是一个助动词,用在一般将来时
    + 强调 **马上,立刻** 一般不需要时间状语
  + how/what about... 怎么样?
    + 一般用来提出建议, 请求, 或者反问
    + how about this watch? - 这个手表怎么样?
    + what about me? - 我怎么样?
    + what about going to the cinema tonight? - 今天晚上去看电影怎么样? (建议)
    + what about you? - 你呢? (反问)

### before - after

1. (时间,顺序)在...之(前/后) - 主要做时间状语
   
   + after/before + 具体的时间 或 一个时间点
     + after + 表示几点钟的数词
     + after 10:00 - 在十点钟之后
   + after/before + 表示具体时间 或 时间点
     + 英语有些词语可以表示时间点
     + after school - 放学后 = after + 放学时间
     + 就如 放学后一样 还有很多词可以代指具体事件
     + after work - 下班后
     + after class - 下课后
     + after the meeting - 散会之后
   + after/before + 岁数的词语 = 几岁后
     + after 35 years old - 35岁之后
   + after/before + 年份的数词 - 几年后
     + after 1999 = 1999年之后
   + after/before + 星期几,节日
     + after Christmas - 圣诞节后
     + after Sunday - 星期天后

2. after + 一段时间 - 一段时间之后
   
   + 只能用在过去式 - 其他时态用 in
   + after + a/an/one + 表时间的名词单数
   + after + 数词 + 表时间的名词复数

+ (位置)在...前面, 当着...的面
  
  + before + 表示**物体,地点,场所或人(代词宾格)**的词语
  + 作用: 地点状语, 后置定语, 表语
  + 相当于 in front of
  + before me - 在我前面
  + before the computer - 在这电脑前面

3. after + 动名词短语
   
   + 强调顺序关系
   + after getting up,... - 起床后
   + after finishing my homework,... - 在完成作业后

4. 做表语, 追赶,追求,追寻
   
   + the police are after the thief - 警察在追小偷
     + 在小偷后面 = 追赶
   + the cat is after a mouse - 这只猫在追益智老鼠

5. 依照,仿照
   
   + after + 名词/代词宾格
   + read after me - 跟我读

6. 习惯用语
   
   + after all - 毕竟
   + he is a child after all - 他毕竟是个孩子

### of

+ 固定搭配
  + think of - 考虑
  + dream of - 梦见
  + hear of - 听见
  + get rid of - 摆脱
  + be proud of - 为...感到自豪
  + be afraid of - 害怕
  + be fond of 喜欢
  + be sike of 厌恶/厌倦

+ 构成量词
  + a glass of water

+ of 作为一个普通介词的时候, 主要意思就是 **...的**
  + of + 无生命的名词 - 形成 名词所有格
  + the price of the book - 书的价格
+ 同样也可以跟 动名词, 代词, 数词 等, 用法不变
  + of them - 她们当中的
  + one of them - 她们中的一个
  + all of us - 我们中所有人
  + of learning English - 学英语的

  + 这个有生命的必须比较长 或 这个名词本身带有一个 后置定语/同位语
  + of __the man__ _near the window_ - 因为带了后置定语无法使用('s)所有格
  + 同位语 - of CYK, the man from HuBei

### to

+ 很多时候 to 不是介词, 而是动词不定式的符号 - 没有意思
  + **to + 动词原型** , 这时候的 to 没有意思
  + 如: I __want to buy__ a book - 我想买一本书
+ 在双宾动词也是一个符号
  + I will tell you a story = I will tell a story to you

+ 固定搭配
  + be allergic to - 对...过敏
  + belong to - 属于 (动词短语)

1. (表示方向)朝向, 到...去
   + to + 一个表示位置的词语, 但是必须跟在一个表示**动作位移**的词语后面作**状语**
     + 常见位移动词 : go, run , walk, swim, turn, look, rush
     + 朝...run/swim/look
     + 一般不能作表语, 偶尔可以做**后置定语**
     + 强调从其他位置转移到指定位置

2. (表示时间, 程度, 结果)至, 在...时刻以前
   + 可以搭配 from 使用, 表是 **从...到...**
   + he works from morning to night

3. (表示目的)为了
   + we sat down to dinner - 我们坐下来吃饭

4. (表示比较)比, 对, 比较
   + 表示比较关系, 一般和介词 by 一起使用
     + by 数词 to 数词
     + we won the game by 5 to 3 - 我们以5比3赢得了比赛

5. (表示对象关系)属于, 对, 给
   + 一般出现在**主系表**后面作**状语**
   + 而且形容词比较特殊
   + 主语 be 某些表语 to 表示人的词语 - 表示对默认如何或怎么样
   + she is very kind to me - 她对我很好

### through

1. (表示时间)从...开始到结束, 在...之中
   + __all through__ the/形容词性物主代词/指示代词 表示时间的名词单数
   + 一般翻译为 - 整..., 全...
   + all through the morning - 整个早上
   + all through the month - 整个月
   + all through my life - 我的一生
   + 表达时间概念, 作 **时间状语**

2. (表示空间)穿过, 通过, 贯穿
   + through + 某些名词 - 强调空间上的穿过
   + 作 状语
   + 与 across 的区别
     + across - 表示在表面穿过
     + through - 表示在空间上穿过
   + through the window - 穿过窗户
   + through the woods - 穿过树林
   + through the tunnel - 穿过隧道
   + 因为穿过是一个动作, 所以一般搭配表示运动的动词
     + go / run / come in / fly / swim / walk / move
   + the plane is flying through the clouds - 飞机正穿过云层飞行
   + light comes in through the window - 光线穿过窗户进来

3. (表示方法, 手段, 原因)通过, 经由, 以...
   + 方式状语 - through + 表示 渠道/途径 的词语
   + through ads - 通过广告
   + through his friend's introduction - 通过他朋友的介绍
   + he succeeded through hard work - 通过努力工作他成功了

4. 结束, 完成
   + are you through asking questions? - 我问完问题了吗?

1. (运动的方向)沿着...向上

   + 与介词 through 一样, 作状语的时候也需要和**运动的动词搭配**

  + be satisfied with
  + talk with

1. (同伴, 同道)和...一起, 偕同
   + with + 人/动物 - 跟...一起

   + she is with me - 她跟我在一起

# 从句 subordinate Clause

+ 原理
  + 一个句子 A (subordinate) 成了 一个整局 B (Main) 的 ingredient

+ 分类, 句子成分除了 谓语 其他都可以
  + 宾语: I saw __that the rabbit ate a carrot__.
  + 形容词/定语/关系 从句
  + 名词 - 主语/宾语/表语/同位语/宾语补语 从句
  + 副词/状语从句

+ 形容词/定语/关系 从句的作用就是限定 性质/范围
  + I have a head which is round.
    + 我有好几个头, 其中一个是圆的
+ 还有非限定性的, 仅做补充信息, 即时删除也不会影响句意
  + I have a head, which is round.
    + 我有一个头, 而这个头是圆的.
  + My head (, which is big,) is useful when it rains
  + 非限定性还能代指一整句话
    + The rabbit ate the carrot, which was not surprising.

## 形容词/定语/关系从句

+ a _tasty_ carrot.

+ 关系词
+ 关系代词
  + that, which, who, whom, whose
  + a carrot __that I bought__.
  + a carrot __that the rabbit ate.__.
  + a teacher __who is a rabbit__.
  + a teacher __whom I saw yesterday__.
  + The food __which the teacher likes__ is a carrot.
  + The teacher __whose favorite food is carrot__ is a rabbit.
+ 关系副词
  + where, why, when = at/for/on which
  + the place __where the rabbit ate the carrot__.
  + the reason __why the rabbit ate the carrot__.
  + That was the day when .

+ that - which
  + that: 先行词是 唯一/多选一 的
  + which: 多选一

## 名词从句

+ 主语/宾语/表语/同位语/宾语补语 从句
+ 主语从句要 以 引导词开头
  + Whether, where, when, how, who, what, that
  + 引导词 that 可能省略

+ 主语 - the fact is obvious
  + __That I like animation__ is apparent.
  + 添加形式主语: it is apparent __that I like animation__.

+ 宾语 - 时态基本要一致, 除了客观事实
  + I saw that she played games in my bedroom.
  + 否定加在主句中
    + I don't think (that) she likes me.
  + 而不是在从句中, 虽然意思是一样的, 但英语中不这么用
    + I think (that) she doesn't like me.

+ be 形容词 that
  + I __am sure that__ there's a book on the table.
    + = __confirm that__
  + I __am happy that__ there's a cap on the bed.
    + ~ __like that__
  + I __am afraid that__ she isn't beside me
    + = __fear that__

+ 表语从句 - 从句属于单数用 is/三单格式
  + The problem _is_ __that__ I am very hungry.
  + it _seems_ that she likes me.
  + it _feels_ that she likes me.

+ 同位语
  + The _fact_ __that she likes me__ didn't surprise me.
  + The _question_ __whether she likes me__ is on my mind.
  + I have no _idea_ __who likes me__.

+ 补语
  + You can call me __what you like__

## 副词/状语从句

1. 时间
   + 在某一时间点前: __before__
     + she was asleep __before I was excited__.
     + __before I was excited__, she was asleep.
   + 在某一时间点之中(当时): __when, while, as__
     + when: 时间点
     + while: 时间段
     + as: 同时
   + 在某一时间点之后: __after__
   + 从之前某一时间点开始算: __since__
   + 直到之后某一时间点: __until__
   + 一...就...: as soon as
   + 下次: the next time
2. 地点
   + __where__ + '强调形式'
   + she is sleeping __where she likes bedroom__
   + 与形容词从句的区别就是, 直接表示地点, 没有进行修饰
   + 强调形式: __wherever, anywhere, everywhere__
     + wherever: 不管 哪里
     + everywhere: 所有 地方
     + anywhere: 任何 地方
3. 比较
   + 更...: 形容词比较级 __than__ 从句
     + she is bigger __than__ another (is).
     + She loves me __more than__ I love her
   + 同样得...: __as ... as ...__
     + she is as tall as me. - 口语表达
       + = she is as tall as I am. - 正式表达
   + 越...越...: __the (比较级) ..., the (比较级)...__
     + __The more__ you practice English, __the more fluent__ you become.
4. 条件(句)
   + 条件句中, 从句总是比主句前一个时态
   + 条件句一般为描述信息, 没有时态的变化, 虚拟语气中才会有
     + 因为时态会将信息确定, 而 条件句是不确定的
   + 引导词 __if, unless, provided, as long as, in case__
   + __if she likes me__, I will sleep with her
   + I'll sleep with her __if she likes me__
5. 让步
   + 即使..., 也...
   + __even__ - 假设, although/(even) though - 实事
   + __no matter__ - 不管,不论/ regardless of
   + even if I see a carrot, I will not eat it.
6. 方式
   + __as if__ 好似/好像 ...
     + I feel good as if I have girlfriends.
   + __as...__ 像...一样
     + Eat the carrot as I do.
     + Leave the carrot as it is.
7. 原因 - because (因果) > since (众所周知) > as (显而易见)
   + because 可以在主句前后, 后两个只能在主句之前
   + I want her __because I like her__.
   + __Since she likes me__, she can sleep with me.
   + As you were not here, we ate the carrot without you.
   + for 也可以表示因果, 构成复合句, 而不是复杂句/从句
     + because 表示直接因果, for 表示推测因果
8. 目的
   + 更好体现目的性
     + can, could, may, might, should, would
   + I save money __so that__ I can buy computer.
   + __so that__ 不能放在句首
9.  结果
    + __so that__ / so...that... / such...that...
    + The temperature was low __so that__ the lake froze.
    + I ate so many carrots for lunch that I felt sick afterwards.
    + so 后接形容词, such 后接 名词

# 句型

## There be

+ 用以表示 : 在什么地方, 什么时候 有(存在)什么人或事物
+ 构成 : __there be 主语(普通名词/不定代词) 地点状语 (时间状语)__
+ 主语复数用 are , 单数用 is
+ 如果有两个或以上主语, 以 be 靠近的主语为标准决定格式, **就近原则**
+ 量词同样适用,如: there _is_ __a vase__ of flowers
+ 过去时 用 : was 和 were
+ 将来时 用 : will be / be going to be
+ 现在完成时 用 : have/has been

+ 否定句
  + there be not(any)/no + 主语 ...
  + there is no dictionary on the desk
  + there are not ten people in the room

+ 疑问句
  + are there ten people in the room?

+ 含有情态动词的 there be 句型, be 保持原型
  + there 情态动词 be 名词 地点状语
  + There can be gold inside your wall
  + may 也许, must 肯定

+ 其他形式, be 保持原型
  + there be 中有时不用 be 而是加点其他东西
  + seem to be - 似乎有
  + happen to be - 碰巧有 - 通常为过去时 happened to be
  + used to be - 过去常常有
  + exist, live, stand (坐落), come (倒装)
    + 这些词语会直接取代 be

## 被动句

+ 主动语态
  + 主语是动作的发起者
  + I teach English every day.
+ 被动语态 - **没有宾语** (双宾动词除外)
  + 主语是动作的承受着
  + 强调客观事实
  + **翻译一般直接翻译为中文的主动语态**
  + English is Learned everywhere.
  + He is helped by me - 他被我帮助 - 我帮助了他
  + the song is sung in China.
    + 不翻译为: 这首歌在中国被唱了, 不符合逻辑
    + 而是: 在中国唱这首歌. 作为主动语态翻译
+ by 是一个介词, 在被动语态里面意思是 被, 但是一般不翻译
  + by 一般接动作的执行者
  + By + 单词(代词), 作状语
  + An apple is eaten.
    + 一个苹果被吃了
  + An apple is eaten by me. 
    + 我吃了一个苹果 (一个苹果被我吃了)

+ 构成方式
  + 助动词 be + **及物动词过去分词** 不及物动词不行
  + 这里的 **助动词 be** 没有意思
  + 时态变化
  + 一般现在: is/am/are 过去分词
    + this song is often sung by children.
  + 一般过去: was/were 过去分词
    + The problem was discussed yesterday
  + 一般将来: will/(be going to) be 过去分词
    + Many buildings will be built in my hometown.
    + she is going to be forgiven tomorrow.
  + 现在进行: is/am/are being 过去分词
    + 注意某些动词不能用于进行时态
    + The movie is being watched.
  + 现在完成: Has/have been 过去分词
    + The letters have been posted
+ 带有情态动词
  + 情态动词 be 过去分词
  + All the questions on the exam paper must be answered
  + This work can be done easily.
+ 双宾动词
  + 可以有有一个宾语
  + I gave you a book.
    + 用直接宾语做主语: You were given a book
  + I gave a book to you
    + 间接宾语做主语: A book was given to you

+ 否定
  + The room was not cleaned.
  + The ball hasn't been played.
  + The chicken wing isn't being baked.
+ 疑问
  + Was the room cleaned? - Yes, it was
  + Has the ball been played?
  + Is The chicken wing being baked?

+ 主动句和被动句可以表示同一个意思
+ for example - 意思相同, 但是侧重点不同
  + 主动: I clean the room every day.
    + 我每天打扫这个房间
  + 被动: The room is cleaned by me every day.
    + 这个房间每天由我打扫

+ 使用
  + 1. 在不知道主语的情况下使用
    + the car was stolen yesterday. 不知道是谁偷的
  + 2. 主语泛指 或 不言自明 的时候
    + the book in the library must not be taken away.
  + 3. 虽然可以说出 主语(动作执行者)
    + 但是动作承受着 才是 谈话的中心
    + 只强调 承受者, 而不关心执行者
    + My computer has been repaired. - 不强调修电脑的人是谁
  + 4. 出于礼貌, 委婉而避免提及自己或对方时
    + You are wished to do it more carefully.
    + The seat is taken. - 这个坐位有人.
  + 5. 用在文章标题, 广告, 新闻等, (常常省略 be)
    + Girls (are) Wanted - 招女工
    + Road (is) Blocked - 道路堵塞
  + 6. 当动作执行者不是人时
    + Hhe house was washed away by the flood.

## 特殊疑问句

+ 疑问词 + 一般疑问句(或类似一般疑问句)
+ 疑问词必须放开头 , 倒装结构

+ 疑问词
  + Where - 在哪里
  + what - 什么
  + why - 为什么
  + whose - 谁的
  + when - 什么时候
  + who - whom(宾格) - 谁
  + which - 哪一个
  + how - 如何, 怎样

### where

+ 做地点状语
  + 一般情况只出现在 主谓宾的句子
  + 主谓宾句子必须有助动词, 也叫倒装结构
  + 一般现在时中 do / does
    + __Where do__ you learn English?
  + 过去时态用 did
    + Where did you eat yesterday?
  + 将来时 will / be going to
    + where be 主语 going to 动词原型
    + Where will you have your wedding next year?
    + Where are you going to have your wedding next year?
  + 现在进行时 is/am/are
    + Where is she drinking red wine?
  + 现在完成时 have / has
    + Where have you slept for ten hours?
    + Where have you seen her?

+ 做表语
  + Where are you?
  + Where will my car be tomorrow?
  + Where will be ...
  + Where have you been (省略了 to) - 你去哪了(你之前去哪里)

### what

+ 做宾语
  + 需要助动词 和 及物动词
  + What do you Know?
  + What have you done to her?
  + What will you tell me? - 双宾动词

+ 表语
  + What is this peculiar thing?
  + What is your plan?

+ 主语


### why

+ 原因状语
  + 主系表, 主谓宾
  + Why did you do it?
  + Why are they screaming?
  + Why are you so sad?

+ 主语 - 何必
  + 表示反问, 表示没必要
  + Why get upset just because you got one bad grade?
  + 何必因为一次成绩不好就想不开呢？

### whose

+ 作定语
  + Whose + 名词(不能是专属名词) - 主谓宾 do / does
    + __Whose book__ _do_ you like?
    + Whose money do you spend?
  + 过去式 did
    + Whose car did he steal?
  + 一般将来时 will / be going to
    + Whose daughter will he marry next year?
    + Whose daughter is he going to marry?
  + 现在进行时
    + Whose food are you tasting?
  + 现在完成时 has / have
    + Whose purse have you taken?

+ 作表语 - 形容词性 - 主系表
  + Whose is this?
  + Whose car is this?
  + Whose is this car?

### when = what time

+ 做时间状语 - 什么时候
  + 一般现在时 - do / does
    + When do you bathe?
    + When do you get up every morning?
    + When 已经是时间状语了, 所以一般不用加
  + 过去 did
  + 将来 will / be going to

+ 作表语
  + When is your birthday?
  + When is next meeting?

### who - whom (宾格)

+ Whom 做宾语
  + 做宾语, 需要搭配 及物动词
  + Whom do you like?
  + Whom did I discouraged?

### which 哪一个

+ 用法如 whose 作定语
  + Which + 可数名词单数
  + Which game do you like?
  + Which movie have you watched?

### how

+ 做表语
  + How are you?
  + How is your wife?
  + How have you been? - 你最近还好吗?

+ 方式状语 - 如何, 怎么样
  + How 助动词 主语 谓语 宾语 - 主谓宾, 不能用主系表
  + How do you learn English?

+ how long - 多长, 多久 - 时间状语, 表语
  + 时间状语需要使用在 现在完成时态的第一种用法
    + How long has she slept?
    + How long have you learned English?
  + 表语, 一般问一个事物的长度
    + How long is your hair? 你的头发有多长?

  + 表语

    + How far is your home from here?

+ How mach/many - 多少
  + 两个意思相同, 但是用法不同
  + How much 可以单独做表语 表示 多少钱, inquire goods price

  + How much 修饰不可数名词
  + How many 修饰可数名词复数

  + 表示频率, 一般出现在 一般现在时态中
  + How often do you go home?
  + How often do you drive?

+ 通常就是吧谓语提前
